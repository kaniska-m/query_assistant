# 🚀 Query Assistant Backend

A production-ready backend for:

- 🔹 SQL Query Executor
- 🔹 Dynamic Query Builder (Form-based)
- 🔹 Oracle Database Integration
- 🔹 Secure Query Validation

---

## 🧠 Features

### ✅ SQL Executor
- Run raw SQL queries
- Safe validation (only SELECT allowed)
- Execution time tracking

### ✅ Query Builder
- Select table, columns
- Add filters (AND / OR / IN / BETWEEN)
- Auto-generate SQL
- Execute queries safely

### ✅ Metadata APIs
- Get tables
- Get columns
- Get distinct values

---

## 🏗️ Tech Stack

- FastAPI
- Oracle DB (oracledb)
- Python 3.10+
